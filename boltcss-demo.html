⚡️Bolt.css

Basic styles for HTML elements. Providing a mostly-reasonable set of styles without classes.

Use it for simple static sites or as a starting point and add your own classes. Don't use it as a complete solution as this doesn't include complex components, form validation, modals, or layout grids.

How I made this

I used the fantastic HTML Elements Reference on MDN and modified as I saw fit.

Features

  • No reset used
  • No layout support
  • Responsive
  • Tiny
  • Extensible
  • Limited theming

How to use

  • Use it as-is to spruce up your static site
  • If you are an experienced front-end developer, use this as a great starting point.
